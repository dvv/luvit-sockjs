# SockJS-luvit

SockJS server in [Luvit](https://github.com/luvit/luvit).

## Installation

    git clone https://github.com/dvv/luvit-sockjs sockjs
    cd sockjs
    lui

These commands will build luvit and necessary libraries (crypto).

## Tests

    cd tests
    ./app.lua
    open http://localhost:8081/index.html
